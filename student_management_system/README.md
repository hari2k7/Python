# Student Management System

A menu-driven Student Management System built using Python.
This project uses file storage (JSON) to persist student records.

## Features
- Add student
- View all students
- Search student by ID
- Update student details
- Delete student
- Persistent storage using JSON

## Concepts Used
- Object-Oriented Programming
- File Handling
- Modular Programming
- CLI Menus

## How to Run
```bash
python main.py
